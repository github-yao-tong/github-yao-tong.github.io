用hexo(yilia主题)+github搭建自己的博客

本文将一步一步的教会你，如何用hexo在github上搭建自己的blog，包教包会，不会再教，一定要把你教会。

github 配置

github的官网网址：https://github.com/， 找到右上角的sign up按钮，不要点击sign in，这个是登录，然后按要求填写你自己的信息，最后点击 create account按钮，这就创建好了账号。

点击右上方➕按钮，选择New repository，Repository name，必须写上____.github.io，空格上添上你刚才写的username

例如你的username是 xiaoming， 那么你的Repository name就是--> xiaoming.github.io

项目建成后，点击settings，向下拉到Github Pages，点击Choose a theme，然后过一会儿回到github page，在浏览器中输入 ____.github.io访问你刚刚创建的页面。（由于国内网络原因，可能访问很慢，也可能访问不了，均属于正常情况哈）

打开终端，输入如下内容，配置git信息

git config --global user.name "_______"
git config --global user.email "_______"

上面空格处，填入你刚才注册的用户名和邮箱，用于向远程仓库。

然后生成密钥SSH key：

ssh-keygen -t rsa -C "________"

空格处填入你的上面填写的邮箱

打开github，在头像下面点击settings，再点击SSH and GPG keys，新建一个SSH，名字随便。

终端中输入

cat ~/.ssh/id_rsa.pub

将输出的内容复制到框中，点击确定保存。

输入ssh -T git@github.com，如果如下图所示，出现你的用户名，那就成功了。

hexo配置

1.安装node.js

Windows： 打开这个网址 https://nodejs.org/en/ ，建议下载LTS版本哈。

mac： 建议用

brew install node

下载完成后，打开终端，输入如下：

node -v

npm -v

看到版本号， 就是安装成功。

由于原本的npm源下载依赖包很慢，所以推荐使用国内镜像源加速：

npm config set registry https://registry.npm.taobao.org

2.安装hexo

打开终端，输入：

npm install hexo-cli -g

安装完后，输入hexo -v，来验证安装是否成功。

然后创建一个blog文件夹，打开终端，cd进这个blog文件夹，输入

hexo init

init后再输入：

npm install

安装必须的依赖

再然后继续终端输入

hexo g

生成静态网页

hexo s

打开本地服务器，在浏览器输入 http://localhost:4000/ ，访问刚刚用hexo创建的本地页面。

按住ctrl+c，关闭本地服务器。

3.在hexo中配置github

打开blog文件夹，blog文件夹下，找到_config.yml文件，为hexo配置github，修改最后一行。

deploy:
    type: git
    repository: https://github.com/_____/_____.github.io
    branch: master

空格处填写你的github用户名

然后在blog文件下，终端输入：

npm install hexo-deployer-git --save

这个依赖包是将文章推送到github上用的。

4.写文章、发布文章

在blog文件夹下，打开终端，输入

hexo new post "____"

空格处，填写你的文章名字。

然后你就会看到在blog/source/_posts目录下有你刚才创建的 ____.md 文件。

在blog文件夹下，终端输入 hexo g 生成静态页面，再输入 hexo g 就可以通过localhost访问到你刚才创建的文章。

ctrl+v 关闭本地服务器后，终端输入：

hexo d

将本地的hexo创建的blog发布到github上，过一会后，你就可以通过 浏览器输入 ____.github.io，访问到你刚才创建发布的博客和文章啦。

5.配置hexo主题—-yilia

blog文件夹下，终端输入：

git clone https://github.com/litten/hexo-theme-yilia.git themes/yilia

将blog文件夹下的_config.yml中的

theme: yilia

这样就改变好主题了，再在终端下输入

hexo clean

然后

hexo g

再然后

hexo d

过一会儿之后，再用浏览器打开 ____.github.io ，就能看到刚换好的主题了。

ps: 写了太长时间了，先这样吧，后面再改进。
